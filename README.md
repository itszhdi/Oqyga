## Oqiga! — Платформа для управления событиями и продажи билетов

Oqiga! — это современная экосистема для организации мероприятий, объединяющая в себе интерактивные карты площадок, удобный личный кабинет пользователя и мощную панель управления для организаторов.

_____________________
#### Основные возможности:

*1. Интерактивные карты мероприятий*

Визуальный выбор: Выбирайте места прямо на схеме зала или площадки.

Динамические цены: Поддержка различных ценовых категорий в зависимости от зоны.

*2. Управление билетами (Для пользователей)*
   
Цифровой кошелек: Храните все свои билеты в одном месте.

История покупок: Просматривайте детали прошедших и будущих событий.

*3. Функционал организатора*
   
Конструктор событий: Создавайте мероприятия, настраивайте даты и описания.

Управление доступом: Отслеживайте свои мероприятия.
______________________________

#### Технологический стек

Backend - Java (Spring Boot), Maven

Client - Flutter

Database - PostgreSQL

DevOps - Docker, Docker Compose

Integrations - WhatsApp Bot API


____________________________________

#### Структура проекта

 OqygaBackend — Ядро системы, API и бизнес-логика.
 
 oqyga_frontend — Клиентское веб-приложение.
 
 compose.yaml — Быстрое развертывание всей инфраструктуры одной командой.
